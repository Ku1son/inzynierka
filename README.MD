# Courser

## O projekcie
EduApp to darmowa platforma e-learningowa łącząca kursy, interaktywne quizy i elementy rywalizacji (rankingi), aby zwiększać motywację do nauki.

### Problem
Brak bezpłatnego, kompleksowego rozwiązania edukacyjnego z nauką, weryfikacją wiedzy i systemem motywacyjnym w jednym miejscu.

### Dla kogo
- uczniowie i studenci
- szkoły i użytkownicy indywidualni
- przyszli twórcy treści edukacyjnych

## Zakres funkcjonalny
- rejestracja i logowanie użytkowników
- przeglądanie i realizacja kursów
- rozwiązywanie quizów
- punkty i rankingi postępów
- panel administracyjny/moderatorski do zarządzania treściami i użytkownikami

## Technologie
- Frontend: Angular
- Backend: Spring Boot
- Baza danych: PostgreSQL (Docker)
- Konteneryzacja: Docker
- Testy: JUnit, Vitest, Postman
- Narzędzia: VS Code / IntelliJ IDEA, GitHub Issues, draw.io
